Архивы с таблицами МОБ и расширенной информацией по добавленным
стоимостям были скачены по ссылке: 
https://stats.oecd.org/Index.aspx?DataSetCode=IOTSI4_2018#
Export -> Related files
NATIODOMINP.zip, VAROW.zip
Архивы распакованы в папке 'data'
Файл с добавленными стоимостями VAROW.csv содержит UTF-16 маркер
последовательности байтов, не воспринимаемый матлабом.
Скрипт utf16Fix.m (/thirdparty) решает проблему


Данные по обменным курсам были выгружены по ссылке:
https://data.oecd.org/conversion/exchange-rates.htm
csv-файл за 2005--2015 гг. находится в папке 'data/EXCHANGE_RATES'


Данные по ВВП были выгружены из БД OECD по ссылке: 
https://stats.oecd.org/Index.aspx?datasetcode=SNA_TABLE1_ARCHIVE#
Предобработанный csv файл находится в папке 'data/GDP'
